Фронтент технологии

Для фронтенд-разработки используются быстрые, производительные, современные, технологичные и удобные фронтенд-фреймворки: Vue, AMP и библиотека React

React

JavsScript библиотека, которая позволяет собрать сложный UI из компонентов

Vue. js

Прогрессивный фреймворк, преднозначенный для создания клиентских интерфейсов

AMP

Технология ускоренных мобильных страниц с открытым исходным кодом

Laravel

PHP-фреймворк, разработанный для создания веб-приложений и сложных сайтов. Он позволяет упрощать аутентификацию, маршрутизацию, кэширование, работу с базой данных, архитектуру приложения.

Yii2

Технология Yii2 представляет собой компонентный PHP фреймворк, имеющий высокую производительность и эффективность. Имеет готовую инструменты для работы аутентификацию, маршрутизацию, кэширование.
