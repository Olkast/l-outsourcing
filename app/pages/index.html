Наш технологический стек

В своей работе мы основываемся на использовании двух языков программирования PHP и JavaScript, которые позволяют решать большинство бизнес-задач различных проектов

Подробнее

Vue, React (JavaScript)

Данные технологии позволяют разрабатывать веб-приложения и создавать удобный интерфейс как с перезагрузкой страниц, так и без нее, это современные технологии для современной гибкой структуры

React Native (JavaScript)

С помощью данной технологии идет разработка мобильных приложений с созданием удобного интерфейса, это кроссплатформенная технология, которая позволяет создавать приложения как на android, так и на iOS

laravel, Yii2 (PHP)

Серверная технология, написанная на языке PHP, которая предоставляет широкий спектр инструментария для приложений, которые могут быть как отдельным API для отдельного фронтенда, или же полноценное приложение всего цикла с фронтендом на обработке бэкэнда для удешевления цены или ненадобности развития отдельного фронтенда

Modx, Opencart (PHP)

Готовое решение для бизнеса с уже частично готовым функционалом, с последующей доработкой под конкретные цели и задачи заказа, это не плохое решение для малого и среднего бизнеса, а также для теста MVP
